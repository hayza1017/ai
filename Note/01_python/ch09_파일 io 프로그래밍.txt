1절_ txt 파일에 데이터 저장하고 불러오기

1.1 write 쓰기
  - write(), writelines()

- w 또는 wt
기존의 파일이 있는 경우 덮어쓰기, 파일이 없는 경우 생성하여 쓰기

- a 또는 at
기존의 파일이 있는 경우 append, 파일이 없는 경우 생성하고 쓰기

- r 또는 rt
mode의 기본값. 파일이 있는 경우 파일을 열고, 파일이 없는 경우 예외 발생

# 한글을 write 하면 cp949로 저장(win)
# with  구문을 사용하면 매번 close하는 코드가 불필요

	- utf-8 / UTF / UTF8 : 한글조합형 _ 주피터노트북에서 기본
	- euc-kf : 한글완성형(믜같은 특수한 글자는 생성되지 않음)
	- co949 : 확장된 한글완성형(믜 O)
		    └ open() 함수 기본 encoding방식, 엑셀에서
