1절. 파이썬 표준 모듈 사용
- 모듈 : 함수, 변수, class 등을 담고 있는 파이썬 코드 파일(*.py)

[ 파이썬 설치하는 방법]

- pythone.org / microsoft store : 파이썬 표준 모듈 200여개가 포함된 파이썬 인터프린터를 install
                                       (sys, re, datetime, time...)
- miniconda : pythone.org버전보다 조금 많은 모듈이 포함된 인터프린터를 install
- anaconda : 몇백개 이상의 패키지, 개발도구
	(IDE tools;jupyter notebook)를 포함한 인터프린터 설치
        (200명 이상의 기업체에서는 유료)

- uv(pip-파이썬 패키지 설치도구, 패키지/프로젝트 관리도구)
	를 이용한 설치
     * 참조 : https://docs.astral.sh/uv/getting-started/installation/
            - powershell -ExecutionPolicy ByPass -c 
	"irm https://astral.sh/uv/install.ps1 | iex(powershell관리자권한)
     * uv설치 후 파이썬 설치
        
     * Set-ExecutionPolicy RemoteSigned -Scope CurrentUser 
     * Get-ExecutionPolicy명령으로 RemoteSigned출력확인
     * powershell -ExecutionPolicy ByPass -c 
	"irm https://astral.sh/uv/install.ps1 | iex" 
     * powershell 닫았다 다시 열기
     * uv python list
     * uv python install 3.11(업계에서는 3.13이상은 잘 안 씀) -
