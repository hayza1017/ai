    1.1 파이썬 모듈
	1. import 모듈명
		모듈한의 함수, 클래스 변수를 모듈명.함수명()으로 사용
	2.
	#     from 모듈(A) import 함수|클래스|변수(B)
		 - A모듈 안의 B만 로드
	#     form 패키지(A) import 모듈(B)
		-A패키지 안의 B모듈을 로드
	#     form 패키지(A).모듈명(B)import함수명(B)
		-A 패키지 안의 B모듈의 B함수만 로드
	#     from time import ctime
 	#     from time import time
	#     from time import ctime, time

	3. import 모듈명 as 별칭
		모듈을 메모리에 로드하고 별칭을 사용
		import numpy as np

	4. 
	▼ from 모듈명 import 함수명 as 병칭 
	   모듈 안에 함수를 메모리에 로드하고 별칭으로 함수 사용
	▼ from 패키지명 import 모듈명 as 별칭
	   모듈명을 별칭으로 사용
	▼ from 패키지명.모듈명 import 함수명 as 별칭

'''

모듈/패키지이나 모듈 안의 함수를 가져오는 방법

import A                - A모듈을 가져옴
import A as B           - A모듈을 B이름으로 가져옴
from A import B         - A모듈안의 B함수(클래스|변수)를 가져옴
from A import B as C    - A모듈안의 B함수를 C이름으로 가져옴

from A import B         - A패키지 안의 B모듈을 가져옴
from A.B import C       - A패키지 안의 B모듈안의 C함수 가져옴
from A import B as C    - A패키지 안의 B모듈을 C이름으로 가져옴
from A.B import C as D  - A패키지 안의 B모듈안의 C를 D이름으로 가져옴

'''

4절. 패키지 설치 및 삭제 

아나콘다 프롬프트에서
- 패키지 설치 : 
pip install 패키지명 [==버전] / conda install 패키지명[==버전]
- 패키지 확인 : 
pip show 패키지명            / conda list 패키지명
- 패키지 삭제 :
 pip unistall 패키지명        / conda remove 패키지명
