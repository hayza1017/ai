3장 데이터 구조 : 여러 데이터를 한 변수에 담는게 목적

1절_리스트 : [타입에 상관없이 0개 이상의 데이터들]
		ex_['서울', '대전', '부산']

	1.1 리스트 생성 : [], list(range(from, to ,step)_나열 가능한 자료)
 
	1.2 기본 정보 조회
		- min(lst), max(lst), sum(lst), len(lst)
		- sorted(lst, reverse+True_내림차순 / reverse사용 하지 않으면 오름차순) : 정렬된 lst를 변환
		- lst.sort(reverse= True) : lst가 정렬된 값으로 변경
		
	1.3 요소 추가
		- lst.append(value), extend([values]), insert(idx, value)

	1.4 인덱싱과 인덱싱 찾기
		- lst(리스트 변수) [인덱스 _ idx]
		- lst.index(찾을 데이터): 데이터가 없을 경우, error
		- 찾을 데이터 in lst : 찾을 데이터가 있는지 여부 반환
		- lst.count(찾을 데이터) : 찾을 데이터가 몇개 인지 숫자 반환

	1.5 원하는 요소 추출하기
		- 슬라이싱 
		- 리스트컴프리헨션(원하는 데이터 추출)

	1.6 요소 수정 : 인덱싱이나 슬라이싱을 이용
		
	1.7 요소 삭제 : pop(), pop(idx), del lst [idx], remove(데이터)

	1.8 복제(깊은 복사) : 복제할 변수 = 원본 데이터.copy()



2절_ 튜플 (리스트와 유사하나 수정 불가)

	-리스트([])와 유사하지만 읽기 전용. 
	 수정이 필요없는 데이터에서 사용
	-수정이 불가함으로 튜플에 데이터 수정, 추가, 삭제 불가능
	-제공되는 함수가 많지 않음.
	-()를 이용해서 생성됨. tuple()함수 이용해서 생성

3절_ 딕셔너리 : {키:값} 키와 값이 쌍
	keys() : 키목록 / valuse() : 값 목록 / items() : 키-값 목록
     읽기 : 변수[키]보다 변수.get(키, 없을경우값)를 사용하여 에러를 줄이자
	추가 / 수정: 변수[키] = 값 / 변수.update(추가하거나 수정할 값들)
	삭제 : 변수.pop(키) / del 변수[키]_해당키값이 없으면 에러
	해당키가 있는지 여부 : 키 in 변수

	- 중괄호{ }를 이요해서 딕셔너리를 생성. 키(key)-값(value)쌍으로
	  성된 자료구조

	- 인덱스를 이용한 참조를 지원 하지 않음
